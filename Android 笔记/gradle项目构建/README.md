# Android gradle简介

在Android开发过程中经常有莫名其妙的报错，刚开始我并不知道为什么。有些报错直接指向了gradle文件，但是也不知道报错的原因。为了工程的需要，我对gradle进行了一些简单的了解。

## 1 gradle的概念

- gradle是一个构建工具
- gradle是一Groovy语言为基础
- gradle主要面向的是java语言

打开Android Studio内置的终端，在输入如下命令查看gradle版本： 

```
gradlew -v
```
